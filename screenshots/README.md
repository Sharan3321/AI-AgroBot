# Screenshots Directory

This directory contains screenshots of the AI-AgroBot application.

## Required Screenshots

Please add the following screenshots to this directory:

1. **landing.png** - Landing page with features
2. **chat.png** - Chat interface with background
3. **login.png** - Login page
4. **register.png** - Registration page
5. **profile.png** - User profile page
6. **admin.png** - Admin dashboard

## How to Take Screenshots

1. Run the application: `python app.py`
2. Navigate to each page
3. Take full-page screenshots
4. Save them in this directory with the names above
5. Recommended size: 1920x1080 or similar

## Note

Screenshots help users understand the application before installation.
